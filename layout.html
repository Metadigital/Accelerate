Layout

A 5 column grid system which defaults to 1140px. On modern browsers this layout is fully responsive.

Global styles for CSS normalize, typography, and layout

Modern

Accelerate is built with the future in mind. We've got sleek styles for a large range of devices from tablet to desktop with progressive fallbacks for older browsers.

Accelerate is built on HTML5 and as such uses some of its elements. Make sure you have set the correct doctyle and use a Html5 shiv to correctly display them in < IE9. We default to using Modernizr.

Consistent

Accelerate normalizes your css using the best bits from Normalize.css, HTML5 Boilerplate and Twitter Bootstrap that makes browsers render all elements including typography more consistently and in line with modern standards.

Responsive

Accelerate uses Flexo, a 5 column (there's really no need for more), fully responsive grid that falls back to fixed width on legacy browsers. Changing the width of the container is as simple as changing one css property.

If you want to force a fixed width just add the class .flexo-fixed to the body.

Responsive helper classes

Accelerate has baked in some helper classes to make it easier to show and hide elements based upon the layout. For example, you might show a <select> element for navigation on mobile layouts, but a <ul> on tablets or desktops.
