Accelerate

A HTML, CSS, and Javascript framework for rapidly developing websites

Welcome to Accelerate

What is it?

Accelerate is a fully responsive, lightweight, mobile-first framework for rapidly developing websites.

Accelerate helps you rapidly build websites without having to worry about cross-browser consistency by ironing out a lot of the bugs so you don't have to.

Download v1.1.2 Accelerate on Github

Features

Accelerate contains lots of clever features that make your life easier:

  • Mobile-First from the ground up
  • Normalized base css to improve cross-browser consistency
  • Responsive 5 column grid
  • Boilerplate styles for many common elements
  • jQuery plugins to perform common tasks and add a little sparkle to your site.
