Boilerplate

Building blocks to allow rapid development.

Reusable componants designed to provide base functionality

Accelerate provides lots of building blocks that are specifically designed to speed up your development by boilerplating common componants from normalized styles to more complicated layouts we've done the work so you don't have to.

Typography Headings, paragraphs and other components

Typography

All the type in Accelerate is em based so it can be scaled easily by changing the font size on the body. This defaults to the browser standard of 16px/1em. and has been used to define all margins and padding within the framework.
